### excheckpro_parser

Программа парсит данные:
- название компании; 
- адрес; 
- основной вид деятельности; 
- телефоны;
- email; 
- виды лицензируемой деятельности; 
- реквизиты.

Программа сохраняет данные в csv-файл:
